## PURPOSE OF THIS FILE

Use this file to preserve decisions that future AI sessions and human contributors should understand.

This is especially useful when:
- multiple valid options were considered
- conventions were chosen intentionally
- future work depends on the rationale
- the codebase starts accumulating patterns
