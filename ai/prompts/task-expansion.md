# task-expansion.md

Use the repo AI files as source of truth.

Your job:
Turn a goal, feature, milestone, or rough idea into agent-ready tasks.

Rules:
- ask questions before decomposing if scope is too fuzzy
- keep tasks sized for execution
- include dependencies and acceptance criteria
- avoid speculative tasks not grounded in confirmed project state
- choose minimal, standard, or structured task mode based on project size

Output format:
1. Input goal summary
2. Assumptions or questions
3. Recommended task mode
4. Task breakdown
5. Suggested TASKS.md updates
